# jupyter-test
Testing Jupyter notebooks

To run in Windows CMD shell:

* python -m venv venv
* venv\Scripts\activate.bat
* pip install -r requirements.txt
* python vectorize_pdf_files.py
* python query_llm.py

To run in a Codespace (Linux):

* python -m venv venv
* source venv/bin/activate
* pip install -r requirements.txt
* python vectorize_pdf_files.py
* python query_llm.py
